CLASS

tcl.lang.TclRuntimeError -- This class is used to report an unrecoverable errors.

CONSTRUCTORS

TclRuntimeError(String errMsg)

ARGUMENTS

String errMsg ()
Error message to print out.

DESCRIPTION

TclRuntimeError
TclRuntimeError is used to report an unrecoverable error inside any Java code that implements the Tcl interpreter or extensions. When a TclRuntimeError is thrown, it stops Tcl execution immediately and prints out the error message, errMsg.
